# Documentation

This directory contains the technical reference for Tab Group Sync.

The **spec** (`.kiro/specs/tab-group-sync/`) is the system of record for requirements, design, and tasks.

## Available Documentation

- **[TECHNICAL.md](TECHNICAL.md)** — Learning guide and technical reference covering architecture, components, state management, communication patterns, file URL sync, fuzzy search, and restore actions
- **[tutorial.html](tutorial.html)** — 18-part interactive walkthrough of building a Chrome extension with TypeScript and React, grounded in this codebase
