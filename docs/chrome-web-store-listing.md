## Full Description

Tab Group Sync automatically backs up your Chrome tab groups to bookmark folders — so you never lose your work when tabs close, Chrome restarts, or you switch devices.

**🛡️ 100% Private — Your data never leaves your browser**
Unlike Workona, Toby, and other tab managers that require accounts and send your browsing data to their servers, Tab Group Sync stores everything as regular Chrome bookmarks. No accounts. No sign-ups. No external servers. No tracking. Cross-device sync works through your existing Google account's bookmark sync — the same way any bookmark works.

**✨ Key Features**

• Automatic Backup — Tab groups sync to bookmark folders continuously
• Cross-Device Sync — Uses Chrome's built-in bookmark sync infrastructure
• Snapshots — Create point-in-time backups and restore them anytime
• Move Groups Across Windows — Relocate entire tab groups between browser windows
• Restore from Bookmarks — Open backed-up groups as new tab groups on any device
• Selective Control — Choose exactly which groups to sync
• Auto-Sync — Optionally auto-enable sync for new tab groups
• Auto-Cleanup — Configurable removal of inactive groups (7-90 days)
• Native Tab Groups — Preserves group names, colors, and structure
• Export/Import — Back up your data as JSON for maximum portability

**🔒 Permissions Explained**
• tabs/tabGroups — Read your tab group names and URLs to create bookmarks
• bookmarks — Create and update bookmark folders for your backed-up groups
• storage — Save your preferences (auto-sync, cleanup settings)
• alarms — Schedule periodic sync that survives browser restarts

**💡 Pro Tip**
Pair with "Search Bookmarks, History and Tabs" by Fannon for powerful search across all your backed-up tab groups.
