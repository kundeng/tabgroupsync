## Full Description

Tab Group Sync automatically backs up your Chrome tab groups to bookmark folders — so you never lose your work when tabs close, Chrome restarts, or you switch devices.

**🗂️ The Only Extension That Syncs Local Files**
Chrome, Edge, and Brave all silently drop file:// URLs from saved tab groups. If you read local documentation, Dropbox-synced books, Zotero PDFs, or offline references in tab groups — they vanish on restart. Tab Group Sync is the only extension that captures and restores them.

**🔄 Cross-Machine Path Mapping**
Local file paths differ across machines (/Users/foo/Dropbox on Mac vs /home/foo/Dropbox on Linux). Tab Group Sync rewrites path prefixes automatically so your file:// bookmarks open correctly on every machine. One-time setup, works forever.

**🛡️ 100% Private — Your Data Never Leaves Your Browser**
Unlike Workona, Toby, and other tab managers that require accounts and send your browsing data to their servers, Tab Group Sync stores everything as regular Chrome bookmarks. No accounts. No sign-ups. No external servers. No tracking. Cross-device sync works through your existing Google/Microsoft account's bookmark sync — the same way any bookmark works.

**✨ Key Features**

• Local File URL Sync — file:// tabs backed up and restored (NEW)
• Cross-Machine Path Mapping — Bidirectional prefix rewriting for Mac/Linux/Windows (NEW)
• One-Click File Restore — Open all file:// tabs from bookmarks with path mapping (NEW)
• Automatic Backup — Tab groups sync to bookmark folders continuously
• Cross-Device Sync — Uses Chrome/Edge's built-in bookmark sync infrastructure
• Snapshots — Create point-in-time backups and restore them anytime
• Move Groups Across Windows — Relocate entire tab groups between browser windows
• Restore from Bookmarks — Open backed-up groups as new tab groups on any device
• Selective Control — Choose exactly which groups to sync
• Auto-Sync — Optionally auto-enable sync for new tab groups
• Auto-Cleanup — Configurable removal of inactive groups (7-90 days)
• Native Tab Groups — Preserves group names, colors, and structure
• Export/Import — Back up your data as JSON for maximum portability

**🔒 Permissions Explained**
• tabs/tabGroups — Read your tab group names and URLs to create bookmarks
• bookmarks — Create and update bookmark folders for your backed-up groups
• storage — Save your preferences (auto-sync, cleanup, path mappings)
• alarms — Schedule periodic sync that survives browser restarts

**💡 Why This Matters**
Every Chromium browser (Chrome, Edge, Brave) has a known bug where file:// tabs in saved tab groups become blank new tabs on restart (Chromium issue #372978298). Edge Workspaces shows them as "workspace unsupported." This extension is the workaround — bookmarks CAN store file:// URLs, and this extension uses that to preserve your local file tabs.
