To open it here

  1. Install Tab Group Sync in this browser.
  2. On the extension in edge://extensions (or chrome://extensions), turn on “Allow access to file URLs.”
  3. Add a path mapping for this machine in the extension’s settings so the file resolves to where it lives locally.
  4. Reopen this link — the extension will load the file directly.

Your file path is read only in your browser (it lives in the link’s # fragment) and is never sent to this site.
