Why am I seeing this?

  1. This tab was part of a synced tab group that contained a local file (file:// URL).
  2. The browser could not open it directly. This usually means "Allow access to file URLs" is not enabled for this extension.
  3. To fix: go to extension settings, find Tab Group Sync, and enable "Allow access to file URLs".
